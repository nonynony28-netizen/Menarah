# 🌟 Menarah Lighting (إضاءة المنارة)

تطبيق ويب عصري وسريع، مصمم لتقديم تجربة مستخدم سلسة وواجهة جذابة تعتمد على تأثيرات الإضاءة المريحة للعين.

## 🚀 التقنيات المستخدمة (Tech Stack)

تم بناء هذا المشروع بأحدث تقنيات الويب:
* **React.js 18:** لبناء واجهات المستخدم التفاعلية.
* **Vite:** كأداة بناء وتطوير فائقة السرعة.
* **Tailwind CSS:** لتنسيق وتصميم الموقع باحترافية وسرعة.
* **Framer Motion:** لإضافة حركات (Animations) سلسة وديناميكية.
* **React Router DOM:** للتنقل بين صفحات الموقع بدون إعادة تحميل.
